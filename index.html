Projects

iCode

Skills Used: JavaScript, React/Redux, Node.js, MongoDB

A forum built with React and Node.js. iCode is meant for new developers to ask questions and get answers from their peers. Users can select from a variety of topics to browse question. All questions and answers can be edited by the user.

HitGoal

Skills Used: JavaScript, jQuery, Node.js, MongoDB

An app built with Node.js and MongoDB that lets users keep track of progress on their goals by regularly leaving updates. Updated goals are noted by a green border on the left-hand side of each individual goal. Previous updates are stored and can be reviewed by hitting the View Updates button.

CryptoPrice

Skills Used: HTML5, CSS3, JavaScript, jQuery

An app that uses the CryptoCompare API to tell you which exchange you can buy your favorite crpyotcurrency at for the cheapest price. Users type in the symbol for a chosen crpytocurrency and a table is displayed that shows important information such as the current price, the high and low for that day, and the price one week prior.
